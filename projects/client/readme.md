### REACT Project Structure 

```
src
├───app         # used to store redux store and reducers

├───components  # shared components

├───features    # feature based modules (e.g. auth, profile, etc.) containing their own components, routes, and redux slices

├───helpers     # helper functions

├───pages       # pages (e.g. home, about, etc.)

├───validations # yup validations schemas
